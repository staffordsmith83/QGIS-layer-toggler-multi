QGIS Layer Toggler Plugin

Toggler the visibility of your layers with keystrokes.

Installation Instructions

  1. Download this plugin as a zip file
  2. Open QGIS
  3. Goto Plugins>Manage and Install Plugins
  4. Click 'Install From Zip'
  5. Enter the path to your downloaded Zip file and click 'Install'
  6. The plugin will be added with its own icon on its own toolbar
Notes:
  • Some shortcuts may not work, if there is a QGIS wide shortcut that overrides it. Go to settings>Keyboard Shortcuts to check.
  • The numeric keypad modifier was removed, as this was buggy in Linux. All number keys now activate the shortcut, when in numeric mode.

For more help see the QGIS documentation: https://docs.qgis.org/3.10/en/docs/index.html
